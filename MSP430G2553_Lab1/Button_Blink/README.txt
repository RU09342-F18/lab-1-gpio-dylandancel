The program makes an LED blink when the button is pressed. The button is set to BIT3. When the button is not being pressed the green LED will be high
resembling a normally open circuit. When the button is pressed the red LED will turn on and the green will turn off the red LED represents a normally closed
Circuit. The manner in which they turn on and off can be adjusted by altering the conditional of the if statement.
