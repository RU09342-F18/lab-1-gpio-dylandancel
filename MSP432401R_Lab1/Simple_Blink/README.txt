This sets a blue LED to blink at a rate that is desired to the user. The direction of pin2 bit1 must be
set as an output. The integer x is used in an incrementer which is modulo'd this is the main control
for which the LED blinks. The frequency at which the LED changes based on the number that is
chosen.
